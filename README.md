# 🎁 Juego de Amigo Secreto

Un divertido juego de "Amigo Secreto" donde los participantes ingresan sus nombres y el sistema selecciona aleatoriamente a un amigo secreto para cada uno.

## 🚀 Características
✅ Permite agregar participantes a la lista.  
✅ Evita la repetición de nombres.  
✅ Permite eliminar nombres antes del sorteo.  
✅ Realiza el sorteo del amigo secreto de forma aleatoria.  
✅ Interfaz amigable y diseño responsivo.

## 🎮 Cómo Jugar
1️⃣ **Abrir `index.html` en el navegador.**  
2️⃣ **Ingresar los nombres de los participantes.**  
3️⃣ **Hacer clic en "Añadir"** para agregarlos a la lista.  
4️⃣ **Eliminar participantes (opcional).**  
5️⃣ **Hacer clic en "Sortear amigo"** para seleccionar un amigo secreto aleatoriamente.  
6️⃣ **Ver el resultado en pantalla.**
